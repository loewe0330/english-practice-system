# 英语知识清单智能练习系统

这是一个基于 Next.js、TypeScript 和 App Router 初始化的英语知识清单智能练习系统。

## 技术栈

- Next.js
- React
- TypeScript
- ESLint

## 本地开发

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

打开浏览器访问 [http://localhost:3000](http://localhost:3000)。

## 可用命令

```bash
npm run dev
npm run build
npm run start
npm run lint
```
